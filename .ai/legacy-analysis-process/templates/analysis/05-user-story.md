# User Story Template

**Usage**: Use this template when generating user stories from requirements.

```markdown
# User Story: {US-ID} - {Title}

**As a** {role},
**I want to** {feature/action},
**So that** {benefit/value}.

## Acceptance Criteria

- [ ] **Scenario 1**: {Title}
  - **Given** {context}
  - **When** {action}
  - **Then** {outcome}

- [ ] **Scenario 2**: {Title}
  - **Given** {context}
  - **When** {action}
  - **Then** {outcome}

## Technical Notes

- **Source Requirement**: {FR-ID}
- **Related Components**: {Component List}
- **Estimated Complexity**: {Story Points / T-Shirt Size}

## Definition of Done

- [ ] Unit tests passed
- [ ] Code reviewed
- [ ] Acceptance criteria met
```
